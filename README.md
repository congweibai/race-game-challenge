# React + TypeScript + Vite

node verion: v23.10.0

## Commands

| name                 | commands      |
| -------------------- | ------------- |
| install dependencies | `npm install` |
| run locally          | `npm run dev` |

## File structure

### general folder structure:

```
src
├── App.tsx
├── __tests__
├── assets
│   └── ...(some images)
├── components
│   └── ...(global reused components go here)
├── helpers
│   └── ...(frontend helpers)
├── hooks
│   └── ...(global hooks go here)
├── index.css
├── mockApis
│   ├── handlers
│   ├── repos
│   ├── services
```
